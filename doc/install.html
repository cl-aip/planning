Downloading and Installing Lisp Code for Automated Planning

This page gives instructions for retrieving the code for the book, and installing it on your local system. This installation procedure need only be followed once.

Downloading the Code

To download the code, follow these steps:

  1. Create a directory where you want the code to reside, and change to that directory. On Unix, this might be "mkdir <planning>; cd <planning>".
  2. Download the planning system. The easiest way is to click on this link to get planning.zip (154KB).
  3. Execute the command, on Unix, unzip planning.zip, or on Windows, drag the planning.zip file to a program such as WinZip. to unpack the archive of files.

Installing a Lisp Interpreter/Compiler

There are many available Lisp compilers that you can use. However, we have tested the code only the following:

  • Franz Allegro Common Lisp for Windows.

We hope that someone ports the program to any other Lisp system.

If you have no Allegro CL system, you can install the evaluation version from Franz Inc. It is free but some months later you will be required to respond the query when starting lisp system whether you upgrade or not. However, you can continue to use it with replying "CONTINUE".

How to change character mode:

We always use ACL in modern mode (character case sensitive), in which lisp system distinguishes upper-case letters and lower-case letters. We recommend to follow the instruction below in order to set up modern mode for the evaluation version. If you purchase ACL, the choice of character modes is just menu selection in Windows version.
  1. Run original aclxx-express.
  2. Execute the following form in Lisp window. You will obtain 'allegro.exe' file under 'C:\Program Files\acl80-express' folder. 
         (progn
       (build-lisp-image "sys:allegro.dxl" :case-mode :case-sensitive-lower)
       (when (probe-file "sys:allegro.exe") (delete-file "sys:allegro.exe"))
       (sys:copy-file "sys:allegro-ansi.exe" "sys:allegro.exe"))

  3. To start Lisp, invoke 'allegro.exe' instead of 'allegro-ansi.exe'. In Windows, it is good idea to set up the start menu for 'allegro.exe'. You may copy the start menu folder for 'allegro-ansi.exe' and revise the link of short-cut file.

Installing the Code

  1. Make sure you have both read-permission and write-permission for the directory where the code is kept.

  2. The defsystem program for planning works depending on the file/directory structure. Please do not change the relative file/directory structure, but you can place anywhere the whole system with keeping the relative file/diretory structure. Then, use the defsystem program to compile, load, etc.

Running the Code

The steps above need be done only once; you are now ready for the easier process of using the code.
